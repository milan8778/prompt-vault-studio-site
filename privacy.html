Privacy Policy

Prompt Vault Studio

Last updated: 2026-06-01

Prompt Vault Studio is a local desktop application for organizing and reusing prompts. This Privacy Policy explains what the app stores and what it does not collect.

Local-first design

Prompt Vault Studio stores app data locally on your device. The app does not require an account and does not provide cloud synchronization.

Data the app may store locally

The app may store locally:

  • prompts you create, edit, import, or save from templates
  • collections, categories, tags, favorites, and trash state
  • variable values and reusable variable presets
  • appearance and language preferences
  • import/export state needed for normal local use

No cloud, analytics, telemetry, or AI API calls

Prompt Vault Studio does not:

  • require a login
  • upload your prompts to a cloud service
  • use analytics
  • use telemetry
  • sell user data
  • include advertising
  • call external AI APIs

Import and export

You can export your local library as a JSON backup file. Exported files are controlled by you. If you share an exported backup, anyone with access to that file may be able to read its contents.

Third-party services

The app is designed to work locally. If you manually copy a prompt into another app or website, that separate service is responsible for its own privacy practices.

# Prompt Vault Studio GitHub Pages site

Upload these files to a public GitHub repository and enable GitHub Pages.

Recommended repository:
`prompt-vault-studio-site`

Recommended Pages source:
`Deploy from a branch` → `main` → `/root`

After GitHub Pages is published, use:

Privacy Policy URL:
`https://<GITHUB_USERNAME>.github.io/prompt-vault-studio-site/privacy.html`

Support URL:
`https://<GITHUB_USERNAME>.github.io/prompt-vault-studio-site/support.html`

Important:
Replace `<SUPPORT_EMAIL_OR_URL>` inside `privacy.html` and `support.html` before final Microsoft Store submission.

Prompt Vault Studio

Private local prompt library for Windows.

Prompt Vault Studio helps you organize, edit, reuse, and copy your best prompts. It is designed as a local-first desktop app: no account, no cloud sync, no analytics, no telemetry, and no AI API calls.

What it does

  • Save and organize reusable prompts.
  • Use collections, categories, favorites, and search.
  • Work with prompt variables such as [topic], [audience], and [tone].
  • Apply reusable variable presets and copy the filled prompt.
  • Import and export your library as a local JSON backup.
